Oral Reports

Using the web as a research tool is often the first step into the internet that teachers and students take. Learning how to research well, to find and recognize quality sitehss, to use the "deep web" are essential skills to teach. This page provides some resources to help teachers teach about web research.

Searching the Web

Searchpath offers an attractive tutorial on websearching, offered through the WMU library.

Ernest Ackerman's Searching and Researching on the Web is tremendous.

Evaluating Information Found on the Web

Elizabeth's Kirk's Evaluating Information Found on the Internet is a useful short resource.

UCLA Library has a useful list of questions for Thinking Critically About WWW Resources.

Kathy Schrock' Guide for Educators helps teachers integrate website evaluation into their classes.

Research Sites Relevant to the Teaching of Literature

David Issacson at the WMU library has created a rich site overview library and web resources for English and American Literature research.

Literature on Line (LION) is a rich electronic resource for literary research, offered through the WMU library.

The University of Virginia Electronic Text Archive is enormous.

The Voice of the Shuttle has rich database of internet resources for literary scholarship.

Project Muse is a collection of electronic academic journals, many focus on literature.
