Literature on the Web
Participation

As an experimental class Literature on the Web calls for unusual flexibility, independence, dedication, and creativity. This class will be run in seminar format and learning will depend, in part, on the involvement of all class members. Class participation includes attendance, participation in class, completion of reading and internet research, and regular written assignments, such as presentations about websites, reading notes, etc. (See syllabus.)
